Mitigating Black Hole Attacks in WSN using Node-Resident Expert System

https://github.com/AJAYLODIPURA

ADIOS based Expert system designed in tcl using NS2 for detecting intrusions and Black Hole Attacks in Sensor Networks.

It is a Robust and Flexible attack detection scheme which uses Watchdog Mechanism and light weight Expert System on each node to detect anomalies and spurious routes of Packet drop attacks like Black Hole Attacks or Sink Hole Attacks.
